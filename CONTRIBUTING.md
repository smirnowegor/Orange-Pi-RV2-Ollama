# Contributing

Thank you for your interest in contributing.

- Open an issue to discuss large changes or feature requests.
- Fork the repo and create a topic branch for your work.
- Keep commits focused and add a clear commit message.
- Add tests where appropriate and update documentation.
- Create a pull request; maintainers will review and provide feedback.

Small PR template checklist:
- [ ] I have read the README and USER_GUIDE
- [ ] My changes are covered by tests or are documentation-only
- [ ] Commit messages are clear
