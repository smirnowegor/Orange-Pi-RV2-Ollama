# Security Policy

## Reporting a vulnerability
If you discover a security issue in this project, please open a **private** GitHub Security Advisory or create an issue labelled `security`. Do not include sensitive details in a public issue.

## Supported versions
This repository contains scripts and documentation for Orange Pi RV2; security fixes will be considered on a best-effort basis.

## Disclosure timeline
Please allow reasonable time for the maintainers to respond. For urgent issues, use a private GitHub Security Advisory.
